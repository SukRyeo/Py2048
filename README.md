# Py2048
This is a game named 2048 designed by Python.  
## INSTALL&RUN
```python
python3 manager.py
```
## REQUESTS
* python3
* os、random、time 库

## FAQ
**Q1：怎么运行程序？**
A1：命令行键入$ python3 manager.py 命令运行

**Q2：怎么操作游戏？**
A2：使用 w、a、s、d、Enter 5个按键操作游戏。

**Q3：怎么反馈Bug？**
A3：发送邮件“反馈Bug+2048+Bug详情”为主题的邮件发送到邮箱
